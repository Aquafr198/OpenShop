# create-openshop

> Scaffold an [OpenShop](https://github.com/Aquafr198/OpenShop) headless Shopify
> storefront in seconds.

## Usage

```bash
npm create openshop@latest
# or
pnpm create openshop
# or
yarn create openshop
```

You'll be prompted for a project name, a template (Next.js or Node) and your
Shopify store domain. The generated project comes wired with
[`@openshop/core`](https://www.npmjs.com/package/@openshop/core) and a ready
`.env`.

Full documentation: [monorepo README](https://github.com/Aquafr198/OpenShop#readme).
